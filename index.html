Typography

In the axial system, the text is divided by a line, otherwise known as the axis. To execute this, I have created diagonal lines and placed text ajacent on both sides.

The radial system consists of text which extends from a single point. Within the centre of the polygon shape i have created is the focal point of my radial system

Dilatational typography consists of text which stays on a circular path. I have created this by having my text follow a path around a polygon in the centre.

Random typography follows no set of rules. text and objects can be pplaced randomly on the canvas.

In the grid system, the text fits into a number of boxes which are arranged in a proportional grid layout. My text follows a 4 box grid layout.

The modular system is similar to the grid system, however the modular structure does not have to follow a uniformed grid layout.

In the bilateral system, text is arranged symetrically on an axis. My text is placed on a central diagonal axis. I have created an uneven shape which i have flipped and place twice on the axis to created symmetry.

Transitional typography follows a wave-like, flowy pattern. Ihave placed my text within wave like layered structures.
